La mission
Ta page profil avec CSS complet
Prends la page HTML que tu as construite au chapitre 1 et ajoute-lui tout le CSS. Pas d'exercice scolaire — une vraie page que tu peux partager sur Instagram ou LinkedIn.
Ce que ton CSS doit avoir
  • Variables CSS dans :root pour toutes les couleurs et espacements
  • Typography — font-family, font-size, line-height avec clamp() pour h1
  • Layout Flexbox sur la navbar et les sections
  • Grille pour les liens ou projets avec auto-fit minmax
  • Transitions et transform sur les cards et boutons au :hover
  • Styles de :focus visibles pour l'accessibilite
  • Un @keyframes d'entree sur le header ou le hero
  • Media queries mobile first
  • Dark mode avec prefers-color-scheme ou [data-theme]
  • Design global coherent — couleurs, espacement, typographie harmonieux
