Le modele de boite.
En CSS, chaque element est une boite rectangulaire. Comprendre le modele de boite — margin, border, padding, content — c'est comprendre comment les elements s'organisent et s'espacent sur la page. C'est fondamental.

Chaque element HTML est une boite rectangulaire.

Le modele de boite CSS definit comment chaque element occupe de l'espace sur la page. Il est compose de quatre zones concentriques : le contenu au centre, puis le padding, puis la border, puis le margin a l'exterieur.

Comprendre ce modele c'est la cle pour comprendre pourquoi les elements se positionnent comme ils le font, et comment controler leur taille et leur espacement avec precision.

Schema visuel — les 4 zones du box model
MARGIN — espace exterieur transparent
BORDER — la bordure
PADDING — espace interieur
CONTENT — le contenu
CSS — Box model complet
.card {
  /* Content — la taille du contenu */
  width: 300px;
  height: auto;

  /* Padding — espace interieur */
  padding: 24px;

  /* Border — la bordure */
  border: 1px solid #EFEFEF;
  border-radius: 12px;

  /* Margin — espace exterieur */
  margin: 16px auto;
}
La taille reelle par defautPar defaut en CSS, width et height ne comptent QUE le contenu. La taille totale = width + padding gauche + padding droit + border gauche + border droite. Une card de width:300px avec padding:24px et border:1px fait en realite 350px de large. C'est souvent source de confusion — on corrige ca avec box-sizing.
margin — l'espace a l'exterieur de l'element.

Margin cree de l'espace entre un element et ses voisins. Il est toujours transparent — il n'a pas de couleur de fond. Tu peux le definir pour chaque cote independamment ou en raccourci.

CSS — margin en detail
/* 1 valeur — tous les cotes */
.box { margin: 20px; }

/* 2 valeurs — haut/bas  gauche/droite */
.box { margin: 16px 24px; }

/* 4 valeurs — haut droite bas gauche (sens horaire) */
.box { margin: 10px 20px 30px 40px; }

/* Cotes individuels */
.box {
  margin-top: 20px;
  margin-right: 0;
  margin-bottom: 20px;
  margin-left: 0;
}

/* Centrer un element bloc horizontalement */
.container {
  width: 740px;
  margin: 0 auto;  /* Le pattern pour centrer ! */
}
margin: 0 autoLe pattern classique pour centrer un element bloc horizontalement. Il faut que width soit defini. 0 = pas de marge en haut et en bas. auto = le navigateur calcule automatiquement des marges egales a gauche et a droite. C'est comme ca que le contenu de Kora est centre sur la page.
⚠️
Margin collapsing
Quand deux elements bloc se touchent verticalement, leurs marges fusionnent — on appelle ca le "margin collapsing". Si le h1 a margin-bottom:32px et le p en dessous a margin-top:16px, l'espace entre les deux sera 32px, pas 48px. La plus grande marge l'emporte. Ca ne se produit pas avec padding, ni horizontalement.
padding — l'espace a l'interieur de l'element.

Padding cree de l'espace entre le contenu et la bordure. Contrairement au margin, il prend la couleur de fond de l'element. C'est ce qui fait que le texte ne colle pas aux bords d'une carte ou d'un bouton.

  padding-left: 14px;

border et box-sizing — la propriete qui change tout.

border definit la bordure d'un element. box-sizing: border-box change le calcul de la taille pour inclure padding et border dans width — ce qui est beaucoup plus intuitif.

CSS — border complet
/* Raccourci : epaisseur  style  couleur */
.card { border: 1px solid #EFEFEF; }
.btn  { border: 2px solid #6B1F2A; }
.err  { border: 1px dashed #FF5F57; }

/* Cotes individuels */
.divider { border-bottom: 1px solid #EFEFEF; }
.accent  { border-left: 3px solid #C4622D; }

/* Coins arrondis */
.card { border-radius: 12px; }  /* Uniforme */
.pill { border-radius: 999px; } /* Pilule */
.top  { border-radius: 12px 12px 0 0; } /* Haut seulement */
CSS — box-sizing : la regle a toujours mettre
/* A TOUJOURS mettre en debut de CSS */
* {
  box-sizing: border-box;
}

/* Avec border-box : */
.card {
  width: 300px;    /* = 300px TOTAL, padding inclus */
  padding: 24px;  /* Ne s'ajoute plus a width */
  border: 1px solid #EFEFEF; /* Idem */
}
/* Taille finale = exactement 300px */
box-sizing: border-box — obligatoireMets toujours * { box-sizing: border-box } en tout debut de ton CSS. Ca rend le calcul des tailles previsible. Quand tu dis width: 300px, l'element fait 300px en tout — padding et border compris. C'est le standard de l'industrie. Tous les frameworks CSS (Bootstrap, Tailwind) l'incluent automatiquement.
