La typographie c'est 50% du design.

Les polices, les tailles, l'espacement — c'est ce qui rend un texte lisible ou penible. CSS te donne un controle total sur comment le texte s'affiche.

Il y a deux types de polices : les polices systeme disponibles sur tous les appareils (Arial, Georgia, Helvetica), et les Google Fonts que tu importes depuis le web pour un rendu identique partout.

CSS — Proprietes de base
body {
  font-family: 'DM Sans', sans-serif;  /* Police + fallback */
  font-size: 16px;               /* Taille de base */
  line-height: 1.6;              /* Hauteur de ligne */
  color: #1A1A1A;                /* Couleur du texte */
}

h1 {
  font-family: 'Fraunces', serif;  /* Autre police pour les titres */
  font-size: clamp(28px, 5vw, 56px); /* Responsive auto */
  font-weight: 300;              /* 100 a 900 */
  letter-spacing: -1px;          /* Espacement des lettres */
}
line-height sans uniteline-height: 1.6 (sans px) est relatif a la taille de la police. Sur un texte de 16px, ca donne 25.6px entre les lignes. C'est le bon ratio pour la lisibilite. Evite les valeurs en px pour line-height — elles ne s'adaptent pas quand la police change de taille.
🔤
clamp() — typographie responsive
clamp(min, preferred, max) cree une valeur qui s'adapte. font-size: clamp(28px, 5vw, 56px) signifie : jamais moins de 28px, jamais plus de 56px, et idealement 5% de la largeur du viewport entre les deux. Kora utilise clamp() pour tous ses titres — c'est ce qui les rend lisibles sur mobile ET sur grand ecran sans media query.
Google Fonts — des centaines de polices gratuitement.

Google Fonts propose plus de 1500 polices gratuites. Pour les utiliser, tu importes le lien dans ton head HTML et tu les utilises en CSS comme n'importe quelle police.

Les polices les plus utilisees sur le web : Inter et DM Sans pour le texte courant. Fraunces, Playfair Display, Cormorant pour les titres avec caractere.

HTML + CSS — Importer Google Fonts
<!-- Etape 1 : dans le head HTML -->
<link href="https://fonts.googleapis.com/css2
  ?family=DM+Sans:wght@300;400;500
  &family=Fraunces:ital,wght@0,300;0,600;1,300
  &display=swap" rel="stylesheet">

/* Etape 2 : dans le CSS */
body {
  font-family: 'DM Sans', Arial, sans-serif;
  /* Arial = fallback si Google Fonts ne charge pas */
}
h1 {
  font-family: 'Fraunces', Georgia, serif;
}
La regle du fallbackToujours finir font-family avec une famille generique : sans-serif, serif, ou monospace. Si Google Fonts ne charge pas (utilisateur hors ligne, blocage...), le navigateur utilise la police systeme la plus proche. C'est le filet de securite de la typographie web.
⚡
display=swap
Le parametre display=swap dans l'URL Google Fonts dit : affiche d'abord une police systeme, puis remplace quand la Google Font est chargee. Sans ca, le texte peut etre invisible pendant le chargement — ce qu'on appelle le FOIT (Flash Of Invisible Text). Avec swap, tu as le FOUT (Flash Of Unstyled Text) — moins ideal mais bien mieux que l'invisible.

CSS — Proprietes texte completes
.texte {
  font-size: 16px;
  font-weight: 400;         /* 100 thin → 900 black */
  font-style: italic;        /* normal | italic | oblique */
  text-align: center;       /* left | right | center | justify */
  text-decoration: underline; /* none | underline | line-through */
  text-transform: uppercase; /* none | uppercase | lowercase | capitalize */
  letter-spacing: 2px;       /* Espace entre les lettres */
  word-spacing: 4px;         /* Espace entre les mots */
  white-space: nowrap;       /* Pas de retour a la ligne */
  text-overflow: ellipsis;   /* ... si texte trop long */
}
Hierarchie typographique — l'oeil sait ce qui est important.

Une bonne typographie cree une hierarchie visuelle claire. L'oeil sait immediatement ce qui est important, ce qui est secondaire, ce qui est du detail. Ca se fait avec des tailles, des graisses et des polices differentes.

La regle de base : maximum 2-3 polices sur un site. Une pour les titres (souvent serif, expressive), une pour le corps du texte (sans-serif, lisible). Kora utilise Fraunces pour les titres et DM Sans pour le texte.

CSS — Systeme typographique complet
/* Base du document */
html { font-size: 16px; }
body {
  font-family: 'DM Sans', sans-serif;
  line-height: 1.6;
  color: #1A1A1A;
}

/* Hierarchie des titres */
h1 { font-size: clamp(28px, 5vw, 56px); font-weight: 700; line-height: 1.1; }
h2 { font-size: clamp(22px, 3vw, 36px); font-weight: 600; line-height: 1.2; }
h3 { font-size: clamp(18px, 2vw, 24px); font-weight: 500; line-height: 1.3; }
p  { font-size: 15px; font-weight: 300; line-height: 1.75; }
small { font-size: 13px; color: #737373; }
.caption { font-size: 11px; letter-spacing: 0.5px; color: #B0B0B0; }
🎯Typographie de ta page profilExercice
Ecris le CSS typographique complet pour ta page profil. L'objectif est de creer une hierarchie lisible et professionnelle avec les bonnes proprietes.
Ton CSS doit inclure
  • font-family sur body avec un fallback (sans-serif ou serif)
  • h1 avec font-size, font-weight, et line-height
  • h2 avec une taille inferieure a h1 et une couleur differente
  • p avec font-size minimum 15px et line-height superieur a 1.5
  • Au moins une propriete supplementaire : text-align, letter-spacing, text-transform...
