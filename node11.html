Connecter le front-end — relier l'API au HTML existant.

Le front-end Kora utilise localStorage. Maintenant on le connecte a la vraie API. Les appels fetch() vers l'API remplacent les operations localStorage.

Token dans localStoragePour commencer : stocker le JWT dans localStorage. Chaque requete API ajoute Authorization: Bearer token. Si l'API repond 401, rediriger vers la page de connexion. En production : migrer vers cookies HttpOnly.
Front-end — Connexion API
// api.js — utilitaire front-end

const API = 'https://kora-api.railway.app/api';

function getToken() {
  return localStorage.getItem('kora_token');
}

async function req(endpoint, options = {}) {
  const token = getToken();
  const res = await fetch(API + endpoint, {
    ...options,
    headers: {
      'Content-Type': 'application/json',
      ...(token ? { Authorization: 'Bearer ' + token } : {}),
      ...options.headers
    }
  });

  if (res.status === 401) {
    localStorage.removeItem('kora_token');
    window.location.href = '/login.html';
    return;
  }
  return res.json();
}

// Connexion
async function login(email, password) {
  const data = await req('/auth/login', {
    method: 'POST',
    body: JSON.stringify({ email, password })
  });
  if (data.token) {
    localStorage.setItem('kora_token', data.token);
    return data.user;
  }
  throw new Error(data.error);
}

// Sauvegarder progression
async function sauvegarderProg(moduleId, statut, score) {
  return req('/progression', {
    method: 'POST',
    body: JSON.stringify({ moduleId, statut, score })
  });
}

// Charger le profil
async function chargerProfil() {
  return req('/me');
}

React — la prochaine etape
Le front-end HTML/CSS/JS fonctionne mais devient complexe. React gere l'etat automatiquement. Next.js (React + SSR) = framework le plus utilise. Le back-end Kora est 100% compatible avec n'importe quel front-end.

Logs erreurs en prod
Utilise Sentry (gratuit pour les petits projets) pour capturer les erreurs en prod. npm install @sentry/node. Sentry t'envoie un email quand une erreur se produit avec la stack trace. Indispensable pour debugger sans etre en permanence sur les logs.
