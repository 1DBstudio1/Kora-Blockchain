Flexbox.
Flexbox est le systeme de layout moderne en CSS. Il resout en une ligne ce qui prenait des dizaines de lignes avant. Centrer, distribuer, aligner — tout devient naturel. C'est l'outil le plus utilise dans le CSS moderne.

Une seule propriete change tout. display: flex.

Flexbox est un systeme de layout unidimensionnel — il organise les elements sur un axe, horizontal ou vertical. Tu actives Flexbox sur un conteneur avec display:flex. Tous ses enfants directs deviennent automatiquement des flex items.

Avant Flexbox (avant 2013), centrer un element verticalement en CSS etait un probleme notorious. Il fallait des hacks avec position:absolute et des calculs de margins. Aujourd'hui c'est 3 lignes.

CSS — Activer Flexbox
/* Activer Flexbox sur le parent */
.container {
  display: flex;
  /* Tous les enfants deviennent des flex items */
  /* Ils s'alignent horizontalement par defaut */
}

/* Direction de l'axe principal */
.horizontal { flex-direction: row; }    /* Defaut */
.vertical   { flex-direction: column; } /* Vertical */

/* LE pattern le plus utilise : centrer parfaitement */
.center-everything {
  display: flex;
  justify-content: center; /* Centre horizontalement */
  align-items: center;    /* Centre verticalement */
  height: 100vh;            /* Besoin d'une hauteur */
}

Avant Flexbox — le hack du centrage vertical
Avant 2013, centrer un element verticalement en CSS demandait : position:absolute, top:50%, left:50%, margin-top:-moitie-hauteur, margin-left:-moitie-largeur. Et encore fallait-il connaitre la hauteur exacte de l'element. Avec Flexbox : display:flex + align-items:center. Ca a change la vie de tous les developpeurs front-end du monde.
justify-content — distribuer sur l'axe principal.

justify-content controle comment les flex items sont distribues sur l'axe PRINCIPAL. Par defaut l'axe principal est horizontal. C'est la propriete pour gerer l'espacement entre les elements.

space-between — le plus utilisespace-between met les elements aux extremites et distribue l'espace egalement entre eux. C'est parfait pour une navbar logo-a-gauche menu-a-droite. space-evenly distribue l'espace egalement partout y compris aux bords. space-around donne a chaque element le meme espace de chaque cote (donc les espaces aux bords font la moitie des espaces internes).
align-items — aligner sur l'axe perpendiculaire.

align-items controle l'alignement des flex items sur l'axe PERPENDICULAIRE a l'axe principal. Si l'axe principal est horizontal, align-items controle l'alignement vertical — et c'est ca qui permet le centrage vertical tant attendu.

CSS — align-items
.container {
  display: flex;
  height: 200px; /* Besoin d'une hauteur pour voir l'effet */
}

/* Les 5 valeurs */
.a { align-items: flex-start; } /* En haut */
.b { align-items: flex-end; }   /* En bas */
.c { align-items: center; }      /* Centre vertical */
.d { align-items: stretch; }     /* Etire sur toute la hauteur (defaut) */
.e { align-items: baseline; }    /* Aligne les textes sur leur baseline */

/* Centrage parfait horizontal ET vertical */
.perfect-center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

/* align-self — sur un item specifique */
.special-item {
  align-self: flex-end; /* Override align-items pour cet item */
}

Le header de Kora en Flexbox
Chaque header de module que tu vois sur Kora est fait avec Flexbox : display:flex, align-items:center, height:56px. Le bouton retour a gauche, l'info au centre (flex:1 pour prendre tout l'espace), le badge KRT a droite. Maintenant que tu sais utiliser Flexbox, tu comprends exactement comment ce header est construit.
flex-wrap, gap et flex — les outils qui completent tout.

flex-wrap permet aux items de passer a la ligne quand il n'y a plus de place. gap cree de l'espace entre les items sans margin. flex sur les items controle comment ils grandissent ou retrecissent.

CSS — flex-wrap, gap et flex
/* flex-wrap — passer a la ligne */
.tags {
  display: flex;
  flex-wrap: wrap; /* Les items passent a la ligne */
  gap: 8px;
}

/* gap — espace entre items (replaces les margins) */
.nav {
  display: flex;
  gap: 20px;          /* H et V */
  row-gap: 12px;      /* Lignes seulement */
  column-gap: 20px;   /* Colonnes seulement */
}

/* flex sur les items */
.sidebar { flex: 0 0 280px; } /* Taille fixe 280px */
.main    { flex: 1; }          /* Prend tout l'espace restant */
.equal   { flex: 1; }          /* Partage l'espace egalement */

/* flex: grow shrink basis */
.item { flex: 1 1 250px; }
/* Peut grandir, peut retrecir, base de 250px */

Contenu principal — prend tout l'espace disponible avec flex:1
flex: 1 — la propriete la plus puissanteflex: 1 est le raccourci pour flex: 1 1 0. Ca signifie que l'element peut grandir (grow:1), peut retrecir (shrink:1), et sa base est 0. En pratique : l'element prend tout l'espace disponible. Si deux elements ont flex:1 dans le meme container, ils se partagent l'espace egalement. C'est comme ca que la barre de progression dans le header de Kora prend toute la place entre le bouton retour et le badge KRT.
