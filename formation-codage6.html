Module 6 — HTML
Les tableaux.
Les tableaux HTML servent a afficher des donnees structurees — des comparaisons, des prix, des stats, des horaires. Pas pour la mise en page comme on le faisait dans les annees 90, mais pour les vraies donnees tabulaires.

Exercice 🎯
Un tableau HTML c'est quatre balises qui s'imbriquent.

<table> contient tout. <thead> pour l'en-tete. <tbody> pour le corps. <tr> pour chaque ligne. <th> pour les cellules d'en-tete. <td> pour les cellules de donnees.

Schema visuel — structure d'un tableau
<table>
<thead>
<tr>
<th> Crypto
<th> Prix
<th> Variation
<tbody>
<tr>
<td> Bitcoin
<td> 85 000$
<td> +3.2%
<tr>
<td> Ethereum
<td> 3 200$
<td> +1.8%
HTML — Tableau complet
<table>
  <thead>
    <tr>
      <th>Crypto</th>

    <tr>
      <td>Bitcoin (BTC)</td>

    </tr>
    <tr>

th vs td<th> c'est pour les en-tetes — le navigateur les met en gras par defaut et les lecteurs d'ecran savent que c'est une colonne/ligne de titre. <td> c'est pour les donnees. La difference est semantique et importante pour l'accessibilite.
colspan et rowspan — fusionner des cellules.

Comme dans Excel, tu peux fusionner des cellules horizontalement avec colspan et verticalement avec rowspan.

HTML — colspan et rowspan
<table>
  <thead>
    <tr>
      <th>Formation</th>
      <!-- colspan=2 : cette cellule occupe 2 colonnes -->
      <th colspan="2">Progression</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <!-- rowspan=2 : cette cellule occupe 2 lignes -->
      <td rowspan="2">HTML</td>
      <td>Module 1</td>
      <td>Complete</td>
    </tr>
    <tr>
      <!-- Pas de premier td car rowspan occupe la place -->
      <td>Module 2</td>
      <td>En cours</td>
    </tr>
  </tbody>
</table>

Les tableaux de comparaison de prix
Les pages de tarification de Stripe, Notion, Shopify — ces beaux tableaux de comparaison "Gratuit / Pro / Enterprise" — sont des tableaux HTML (ou parfois des grids CSS). Le HTML de base c'est exactement ce qu'on vient d'apprendre. C'est le CSS qui les rend beaux avec des couleurs, des bordures et des ombres.
Construis le tableau de comparaison des offres Kora.
🎯Tableau de comparaison des offresExercice
Kora a trois offres : Gratuit, Pro et Enterprise. Tu dois creer un tableau HTML qui les compare sur 3 criteres : le prix, le nombre d'articles par jour, et l'acces aux analyses.
Le tableau doit avoir
  • Une balise <table> avec <thead> et <tbody>
  • Un en-tete avec <th> pour chaque colonne
  • 3 lignes de donnees avec <tr> et <td>
  • Au moins un colspan ou rowspan
