CSS propose cinq formats de couleurs differents.

Tu peux ecrire une couleur en hexadecimal, en RGB, en HSL, ou simplement avec son nom anglais. Chaque format a ses avantages — certains sont plus lisibles, d'autres plus pratiques a manipuler programmatiquement.

Hexadecimal
#6B1F2A
RGB
rgb(107,31,42)
RGBA
rgba(...,0.5)
HSL
hsl(350,55%,27%)
CSS — Les 5 formats de couleurs
/* 1. Nom anglais — simple mais limite */
h1 { color: red; }

/* 2. Hexadecimal — le plus utilise */
h1 { color: #6B1F2A; }
p  { color: #333; }     /* Raccourci = #333333 */

/* 3. RGB — rouge vert bleu (0-255) */
h1 { color: rgb(107, 31, 42); }

/* 4. RGBA — RGB + transparence (0 a 1) */
.overlay { background: rgba(107, 31, 42, 0.5); }

/* 5. HSL — teinte saturation luminosite */
h1 { color: hsl(350, 55%, 27%); }
Quel format utiliserHexadecimal pour les couleurs fixes — c'est ce que les designers te donnent. RGBA quand tu as besoin de transparence. HSL pour generer des variantes : meme teinte, luminosite differente pour une version plus claire ou plus sombre.

La transparence avec rgba et opacity.

Deux facons de rendre un element transparent en CSS. rgba pour la couleur seulement, opacity pour tout l'element y compris son contenu. La difference est importante.

CSS — rgba vs opacity
/* rgba — transparence sur la COULEUR seulement */
.overlay {
  background: rgba(107, 31, 42, 0.7);
  /* Le texte a l'interieur reste 100% opaque */
}

/* opacity — transparence sur TOUT l'element */
.ghost {
  opacity: 0.5;
  /* Le texte ET le fond sont a 50% transparent */
}

/* Cas reel — header verre depoli */
.header {
  background: rgba(255, 255, 255, 0.95);
  backdrop-filter: blur(12px);
  /* Effet verre depoli — c'est le header de Kora ! */
}

La propriete background — bien plus qu'une couleur.

background en CSS peut accueillir une couleur, une image, un gradient. Et des proprietes complementaires controlent comment ca s'affiche, se repete, se positionne.

CSS — Background complet
/* Couleur de fond */
body { background-color: #FAFAFA; }

/* Image de fond */
.hero {
  background-image: url('cover.jpg');
  background-size: cover;      /* Remplit tout le conteneur */
  background-position: center; /* Centre l'image */
  background-repeat: no-repeat;
}

/* Raccourci — tout en une ligne */
.hero {
  background: url('cover.jpg') center/cover no-repeat;
}

/* Couleur + image par-dessus */
.hero {
  background:
    linear-gradient(rgba(0,0,0,0.4), rgba(0,0,0,0.4)),
    url('cover.jpg') center/cover no-repeat;
}
📸
background-size: cover
cover est la valeur la plus utilisee pour les images de fond. Elle dit : agrandis ou reduis l'image pour qu'elle couvre tout le conteneur sans deformation. contain dit le contraire : montre l'image entiere meme si ca laisse des espaces vides. Pour les sections hero avec image de fond, c'est toujours cover.
Les gradients — des transitions de couleurs en CSS pur.

Les gradients CSS permettent de creer des degrades de couleurs sans image. Linear-gradient pour un degrade lineaire, radial-gradient pour un degrade circulaire.

CSS — Gradients
/* Linear gradient */
.btn {
  background: linear-gradient(to right, #6B1F2A, #C4622D);
}

/* Avec angle — le gradient de Kora */
.kora-grad {
  background: linear-gradient(135deg, #6B1F2A 0%, #C4622D 50%, #E8956D 100%);
}

/* Radial gradient — cercle */
.circle {
  background: radial-gradient(circle, #FFD700, #FF8C00);
}

/* Gradient de texte — technique avancee */
.gradient-text {
  background: linear-gradient(135deg, #6B1F2A, #E8956D);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
  /* Tous les titres Kora utilisent ca ! */
}
